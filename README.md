# Flask Full-Stack

## Backend
- Python 3.10+
- Create venv: `python -m venv .venv && source .venv/bin/activate`
- Install: `pip install -r requirements.txt`
- Run: `flask run` (PORT=5000)

## Frontend
- `cd frontend && npm i && npm run dev`

## Env
- Copy `.env.example` -> `.env`
- Do not commit `.env`
